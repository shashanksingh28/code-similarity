Important

How do I vote?

On top of each recommendation, you will see stars. On a scale of 1 to 5, give 1 star to a recommedation that was somewhat related to your task to 5 stars if a recommendation really helped you complete your task.

How do I customize the recommendations?

You will see sliders for each type of feature below your code editor. Try reducing or increasing weights of these features and observe change in recommendations.

What are signature features?

  • Return type: Match the type of variable returned. (int, double, String, etc)
  • Modifier: Match the type of modifier. (public, private, static)
  • Arguments: Match the type of arguments. for e.g: if your method takes in a String and an integer as a parameter, try to find a another function that also takes the same.

What are structural features?

  • Statements: Match statements like for, while, if, etc
  • Expressions: Match expressions like boolean expressions, greater than, less than, etc
  • Exceptions: Match exceptions thrown and handled
  • Annotations: Match annotations like '@Override'
  • Methods : Match method calls like Collections.sort(), System.out.println(), etc

What are concept features?

    High level features like:
  • Recursion
  • Casting
  • Polymorphism
  • Exception Handling
  • Inner Classes and Methods

What are language features?

Match text from:
  • Method name
  • Java Documentation
  • Comments
  • Variable names
Help

I am getting a message saying 'Check for compilation errors'. How do I know the error?

Unfortunately the system is not advanced enough to show you the exact error.
Most common causes are:
  • You are missing a semi-colon after some line.
  • You have written an incorrect signature, like 'void double method(){}'. It can return either void or double.

Submit button is disabled?

Submit button will be enabled based on 2 pre-requisites:
  1. You have provided at least 15 votes. You can see the number of votes on the top right
  2. You have customized your recommendations at least once.

Getting irrelevant recommendations?

Try:
  1. Adding more content to your method. This could be comments, code, the way you would naturally do without recommendations.
  2. Check for what kind of features you are looking for and adjust weights accordingly.
